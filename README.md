# opengl-game-two-player-3d-game
* 홍익대학교 컴퓨터 그래픽스 기말 과제

## 개발 환경
- visual C++
- OPENGL

## 게임 설명
* 두명의 플레이어가 필드에 있는 몬스터를 처치하고 떨어진 아이템 박스에서 무기를 획득하여 상대방을 공격하거나 신전 안에 있는 구슬을 더 많이 획득한 플레이가 승리하는 게임입니다.
